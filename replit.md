# AJN — مجموعة علي جان

Full-stack luxury Arabic RTL platform for Ali Jan Group in Tuz Khurmatu, Iraq. Covers e-commerce, services booking, admin dashboard, accounting, inventory, delivery management, order tracking, and gallery.

## Run & Operate

- `npm run dev:backend` — run the API server (port 8080)
- `npm run dev:frontend` — run the frontend
- `npm run typecheck` — full typecheck across all packages
- `npm run build` — typecheck + build all packages
- `npm run codegen` — regenerate API hooks and Zod schemas from the OpenAPI spec
- `npm run db:migrate` — run DB migrations when you intentionally want to apply them
- Required env: `DATABASE_URL` — Postgres connection string, `SESSION_SECRET`

## Stack

- npm workspaces, Node.js 24, TypeScript 5.9
- Frontend: React + Vite + react-router-dom + Framer Motion + Recharts + Zustand (cart)
- API: Express 5
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- API codegen: Orval (from OpenAPI spec)
- Build: esbuild (CJS bundle)
- UI: Tailwind v4 + shadcn/ui components, Cairo font (Arabic), dark-only theme

## Where things live

- `lib/api-spec/openapi.yaml` — single source of truth for API contract
- `lib/api-client-react/src/generated/` — generated TanStack Query hooks
- `lib/api-zod/src/generated/` — generated Zod schemas
- `lib/db/src/schema/` — Drizzle ORM table definitions
- `artifacts/api-server/src/routes/` — Express route handlers (one file per domain)
- `artifacts/ajn-platform/src/pages/` — React pages (public + admin)
- `artifacts/ajn-platform/src/lib/cart-store.ts` — Zustand cart (persisted)
- `artifacts/ajn-platform/src/index.css` — black/gold dark theme tokens

## Architecture decisions

- Contract-first OpenAPI: all endpoints defined in `openapi.yaml`, hooks/schemas generated by Orval — no manual fetch calls
- Zustand cart persisted to localStorage for cart state across sessions
- Admin panel lives at `/admin/*` with its own sidebar layout (no navbar/footer)
- Delivery agent panel at `/delivery` — shows shipped orders, marks delivered
- All prices stored as numeric strings in DB, parsed to float in API responses
- Session auth is minimal (phone-based login, no JWT); auth state stored in React Query cache

## Gotchas

- `react-router-dom` and `zustand` must be installed in `artifacts/ajn-platform` (not root)
- API routes are prefixed `/api` by the reverse proxy; server handles full path
- Admin layout uses `<Outlet>` — all admin child routes render inside it
- Sidebar collapse state is local React state only (not persisted)
- DB numeric fields (price, amount) stored as text/decimal in Drizzle, always parse with `parseFloat()`
